Quick Start Code

Here's how to use the Prediction Cone Menu in your own project: 

import { createPredictionConeMenu } from 'prediction-cone';

// Create menu instance
const menu = createPredictionConeMenu({
  items: [
    { id: 'home', label: '🏠 Home' },
    { id: 'profile', label: '👤 Profile' },
    { id: 'settings', label: '⚙️ Settings' },
    { id: 'logout', label: '🚪 Logout' },
  ],
  ringRadius: 120,
  itemSize: 60,
  coneHalfAngleDeg: 22.5,
  deadzone: 30,
});

// Attach to an element or document
menu.attach(document);

// Handle selection
menu.on('select', (e) => {
  console.log('Selected:', e.item?.label);
});

// Open programmatically
menu.openAt(window.innerWidth / 2, window.innerHeight / 2);

API Overview

Methods:
  • attach(element, options?) — Attach menu to DOM element
  • openAt(x, y, context?) — Open menu at screen coordinates
  • close() — Close the menu
  • setItems(items) — Update menu items
  • on(event, handler) — Listen to events (open, close, change, select)

Theming

The menu uses CSS custom properties for styling. You can customize colors, borders, shadows, and more. Open the browser DevTools to inspect the --pcm-* variables.

Available CSS variables:
  • --pcm-bg — Item background color
  • --pcm-text — Text color
  • --pcm-activeBg — Active item background
  • --pcm-activeText — Active item text color
  • --pcm-border — Border color
  • --pcm-shadow — Box shadow
